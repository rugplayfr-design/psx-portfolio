Space Adventure UI

This project involved creating a comprehensive UI system for a space exploration game. The interface includes animated menus, an inventory system with drag-and-drop functionality, interactive maps with waypoints, and a custom health/stamina indicator. All elements were designed with a futuristic aesthetic that matches the game's theme.

Technologies used: Lua, Roblox Studio, Photoshop

Horror Game Interface

A suspenseful UI for a horror game with dynamic elements that react to in-game events. The health indicator becomes more distorted as the player takes damage, and the inventory system uses a limited visibility approach to increase tension. The puzzle interfaces are designed to be intuitive yet challenging, with subtle visual and audio feedback.

Technologies used: Lua, Roblox Studio, After Effects

E-Commerce System

A complete in-game shopping system with multiple currency support, inventory management, and animated transitions. The system includes a storefront UI, shopping cart functionality, purchase history, and admin controls for managing products. All transactions are securely handled through Roblox's APIs with appropriate error handling and user feedback.

Technologies used: Lua, Roblox Studio, Roblox APIs
