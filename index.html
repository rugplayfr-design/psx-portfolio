Horror Game Interface

A suspenseful UI for a horror game with dynamic elements that react to in-game events. The health indicator becomes more distorted as the player takes damage, and the inventory system uses a limited visibility approach to increase tension. The puzzle interfaces are designed to be intuitive yet challenging, with subtle visual and audio feedback.
